# Telostats Tiles Server

## Dev

```bash
$ cd telostats-tiles
$ mkvirtualenv telostats-tiles
$ pip install -r requirements.txt
```

Ensure an updated version of redis is installed. Then simply run:

    foreman start -f Procfile.dev
